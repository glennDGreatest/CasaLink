## Responsive Design

### Desktop View (>768px)
- 4 payment method cards in a row
- 2-column form layout (Date & Amount side-by-side)
- Full-width text areas
- Modal width: 600px

### Tablet View (600px - 768px)
- 2 payment method cards in a row
- 2-column grid for bank details
- Stacked form fields
- Modal full width with padding

### Mobile View (<600px)
- 2 payment method cards in a row (stacked vertically)
- 1-column form layout (all fields full width)
- Stacked bank details
- Modal full width minus padding
